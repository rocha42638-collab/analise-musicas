# Projeto de Análise de Dados Musicais

## Objetivo
Este projeto tem como objetivo analisar dados musicais e verificar quais características das músicas estão mais relacionadas com a popularidade.

## Base de dados
A base possui informações como:
- Danceability
- Energy
- Loudness
- Valence
- Tempo
- Acousticness
- Instrumentalness
- Liveness
- Speechiness
- Popularity

## Análises realizadas
Foram realizadas as seguintes análises:
- Gráficos de dispersão (Scatter Plot)
- Histogramas
- Boxplot
- Análise de correlação
- Relação entre características musicais e popularidade

## Ferramentas utilizadas
- Python
- Pandas
- Matplotlib
- Numpy
- Jupyter Notebook

## Arquivos do projeto
- analise_disco.ipynb → Notebook com as análises
- base_tratada.xlsx → Base de dados tratada
- pasta graficos → Gráficos gerados

## Conclusão
O projeto mostra, por meio de gráficos e análises, como algumas características das músicas podem influenciar sua popularidade.

Este projeto foi desenvolvido para fins de estudo de Análise de Dados com Python.
